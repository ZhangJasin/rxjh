# Excel配置表格式转换说明

## 📋 文件格式

已将配置表格式从 `.xlsx` 改为 `.xls` (Excel 97-2003 格式)

**文件位置**: `data/cfgcsv/Compound.xls`

## 🔄 如何转换为标准xls格式

由于我创建的是制表符分隔的文本文件,您需要按以下步骤操作:

### 方法1: 使用Excel转换 (推荐)

1. **打开文件**
   - 双击 `data/cfgcsv/Compound.xls`
   - 或者在Excel中: 文件 → 打开 → 选择该文件

2. **数据导入向导**
   - Excel会检测到这是制表符分隔的文件
   - 选择"分隔符号"
   - 勾选"Tab键"和"空格"
   - 点击"完成"

3. **另存为xls格式**
   - 文件 → 另存为
   - 保存类型选择: **Excel 97-2003 工作簿 (*.xls)**
   - 文件名: `Compound.xls`
   - 点击"保存"

### 方法2: 使用WPS转换

1. 用WPS表格打开 `Compound.xls`
2. 文件 → 另存为
3. 选择格式: **Excel 97-2003 文件(*.xls)**
4. 保存

### 方法3: 在线转换工具

1. 访问在线转换网站 (如: https://www.zamzar.com/)
2. 上传当前的 `Compound.xls` 文件
3. 选择输出格式: XLS
4. 下载转换后的文件

## ⚠️ 注意事项

### 格式要求
1. **第一行必须是表头**: ID, 一级分组, 二级分组, ...
2. **分隔符**: 使用Tab键分隔(制表符)
3. **空值处理**: 不需要的材料填0或留空
4. **数字格式**: ID、数量等必须是数字,不能是文本
5. **文本格式**: 分组名称、说明等用文本格式

### 货币类型
- `0` = 无货币消耗
- `1` = 金币
- `2` = 银币
- `3` = 钻石

### 材料配置规则
- 至少需要1种材料(消耗道具1)
- 最多支持3种材料
- 不需要的材料填0
- 材料数量必须>0

## 📞 常见问题

### Q: Excel提示文件格式不匹配?
A: 这是正常的,因为我们创建的是制表符分隔的文本文件。按照上面的"方法1"操作即可正确转换。

### Q: 为什么不能用.xlsx格式?
A: 项目要求使用Excel 97-03格式(.xls),这可能与项目的配置表转换工具兼容性有关。

### Q: 转换后Lua配置需要修改吗?
A: 不需要。Lua配置文件(`game_config/cfgcsv/Compound.lua`)已经是正确的格式,无需修改。

### Q: 如何添加新的合成配置?
A: 
1. 在Excel文件中添加新行
2. 填写所有必填字段
3. 保存文件
4. 更新Lua配置文件(手动或通过转换工具)
