# 合成系统网络请求实现说明

##  网络请求方式

合成系统**完全参考武勋系统**的网络请求方式，不自建网络接口。

## 🔧 使用的API

### 客户端 → 服务端

```lua
-- 统一的自定义消息接口
ssrMessage:sendmsgEx(moduleName, methodName, msgData)
```

**参数说明**:
- `moduleName`: 模块名称 = `"compound"`
- `methodName`: 方法名称 = `"CompoundRequest"`
- `msgData`: 消息数据 = `{compoundID = 1}`

**示例**:
```lua
-- 请求合成
ssrMessage:sendmsgEx("compound", "CompoundRequest", {compoundID = 1})
```

### 服务端 → 客户端

```lua
-- 推送数据到客户端
Message.sendmsgEx(actor, moduleName, methodName, data)
```

**示例**:
```lua
-- 推送合成界面初始化数据
Message.sendmsgEx(actor, "compoundMain", "initCompoundData", {
    group1List = {"材料合成", "特殊合成"},
    group2Map = {...}
})

-- 推送合成成功通知
Message.sendmsgEx(actor, "compoundMain", "CompoundSuccess", {
    targetItemID = 10001,
    targetCount = 1,
    success = true
})
```

## 📋 完整请求流程

### 1. 打开合成界面

**客户端调用**:
```lua
-- 请求打开合成界面
ssrMessage:sendmsgEx("compound", "OpenCompoundUI")
```

**服务端处理**:
```lua
-- compoundMain.lua
function compoundMain.OpenCompoundUI(actor)
    -- 获取分组数据
    local group1List = compoundMain.GetGroup1List()
    local group2Map = {...}
    
    -- 推送数据到客户端
    Message.sendmsgEx(actor, "compoundMain", "initCompoundData", {
        group1List = group1List,
        group2Map = group2Map,
    })
    
    -- 打开UI界面
    sendmymsg(actor, ssrNetMsgCfg.OpenUI, 0, 0, 0, tbl2json({
        uiName = "A_Compound",
        viewName = "compoundMain"
    }))
end
```

**客户端接收**:
```lua
-- compoundMainData.lua 或 compoundMain.lua
function compoundMainData:initCompoundData(data)
    self._group1List = data.group1List
    self._group2Map = data.group2Map
    -- 更新UI显示
end
```

### 2. 执行合成

**客户端调用**:
```lua

    ssrMessage:sendmsgEx("compound", "CompoundRequest", {
        compoundID = compoundID
    })
end
```

## 🗂️ 消息路由注册

### 客户端注册

```lua
-- 在客户端初始化文件中注册
ssrMessage:RegisterNetMsg(ssrNetMsgCfg.compound, compoundMainData)
```

### 服务端注册

```lua
-- compoundMain.lua 文件顶部
Message.RegisterNetMsg(ssrNetMsgCfg.compound, compoundMain)
```

## 🎯 与武勋系统对比

| 项目 | 武勋系统 | 合成系统 |
|------|---------|---------|
| 模块名 | `"wuxun"` | `"compound"` |
| 消息ID | `USER_MESSAGE_ID (1000000)` | `USER_MESSAGE_ID (1000000)` |
| 请求方式 | `ssrMessage:sendmsgEx()` | `ssrMessage:sendmsgEx()` |
| 推送方式 | `Message.sendmsgEx()` | `Message.sendmsgEx()` |
| 注册方式 | `Message.RegisterNetMsg()` | `Message.RegisterNetMsg()` |

## ✅ 优势

1. **统一架构**: 与武勋系统保持一致
2. **无需新增消息ID**: 复用 `USER_MESSAGE_ID (1000000)`
3. **易于维护**: 相同的代码模式和调试方式
4. **自动路由**: 框架自动分发到对应模块和方法
