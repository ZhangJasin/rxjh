# 宠物伤害结算属性说明

## 修改概述

按照新的伤害结算逻辑，修正了宠物伤害相关的属性处理。

## 伤害流程

```
宠物攻击流程：
├─ 宠物攻击怪物 → b_base → 165对怪伤害（固定值）
├─ 宠物攻击玩家 → b_base → 67PK加成（万分比）
│
怪物攻击宠物流程：
├─ 怪物攻击宠物 → m_base → 56对怪防御（固定值减免）+ 116受怪减伤（万分比减免）
│
玩家攻击宠物流程：
├─ 玩家攻击宠物 → base → 68PK减免（万分比减免）
```

## 属性说明

| 属性ID | 名称 | 作用 | 生效场景 | 计算方式 |
|--------|------|------|----------|----------|
| 56 | 对怪防御 | 减免怪物伤害 | 怪物→玩家/宠物 | 固定值减免 |
| 116 | 受怪减伤 | 减免怪物伤害 | 怪物→玩家/宠物 | 万分比减免 |
| 165 | 对怪伤害 | 增加对怪伤害 | 玩家/宠物→怪物 | 固定值追加 |
| 67 | PK加成 | 增加对玩家伤害 | 玩家/宠物→玩家 | 万分比追加 |
| 68 | PK减免 | 减免玩家伤害 | 玩家→玩家/宠物 | 万分比减免 |

## 文件修改

### 1. Market_Def/QFunction-0.lua

#### m_base 函数（怪物攻击）
- **原有逻辑**：怪物攻击玩家时，处理56和116属性
- **新增逻辑**：怪物攻击宠物时，从宠物主人属性获取56和116进行伤害减免
- **代码位置**：第1176-1204行

```lua
-- 对怪减伤和对怪防御（目标是玩家或宠物时生效）
if isplayer(target) then
    -- 玩家对怪防御（属性56）：固定值减免
    -- 玩家受怪减伤（属性116）：万分比减免
elseif targetinfo(target, "RACE") == 6 then
    -- 宠物对怪防御（属性56）：固定值减免，从主人属性获取
    -- 宠物受怪减伤（属性116）：万分比减免，从主人属性获取
end
```

#### b_base 函数（宠物攻击）
- **原有逻辑**：仅计算基础伤害
- **新增逻辑**：
  - 宠物攻击怪物：从主人属性获取165对怪伤害（固定值追加）
  - 宠物攻击玩家：从主人属性获取67PK加成（万分比追加）
- **代码位置**：第1228-1256行

```lua
-- 宠物攻击伤害结算
local masterId = targetinfo(actor, "MASTERID")
local isTargetMonster = not isplayer(target) and targetinfo(target, "RACE") ~= 6
local isTargetPlayer = isplayer(target)

if isTargetMonster then
    -- 宠物攻击怪物：165对怪伤害（从主人属性获取）
elseif isTargetPlayer then
    -- 宠物攻击玩家：67PK加成（从主人属性获取）
end
```

#### base 函数（玩家攻击）
- **原有逻辑**：仅计算基础伤害
- **新增逻辑**：玩家攻击宠物时，从宠物主人属性获取68PK减免
- **代码位置**：第1153-1160行

```lua
-- 玩家攻击宠物：68PK减免（从宠物主人属性获取）
if mastertId and mastertId > 0 then
    local pkReduce = tonumber(abil(mastertId, 68)) or 0
    if pkReduce > 0 then
        local reduced = math.floor(result * pkReduce / 10000)
        result = math.max(1, result - reduced)
    end
end
```

## 属性来源

所有宠物相关属性均从**宠物主人**获取：
- 宠物的56、116、67、68、165属性 = 主人的对应属性
- 通过 `targetinfo(actor, "MASTERID")` 获取主人ID
- 通过 `abil(masterId, 属性ID)` 获取主人属性值

## 计算公式

### 固定值计算
```lua
最终伤害 = 基础伤害 ± 固定值
```

### 万分比计算
```lua
追加/减免伤害 = floor(基础伤害 * 属性值 / 10000)
最终伤害 = 基础伤害 ± 追加/减免伤害
```

### 最低伤害
所有减免后的伤害最低为 **1**：
```lua
result = math.max(1, result - reduced)
```

## 注意事项

1. **BattleManager中的处理**：67和68属性的公式模板已添加到CustomPassiveTemplate.lua，可以在BattleManager的配置中引用
2. **属性传递**：宠物的所有战斗属性均从主人获取，确保主人属性变化后宠物即时生效
3. **RACE=6**：宠物的种族标识，用于判断目标是否为宠物
4. **MASTERID**：通过 `targetinfo(petId, "MASTERID")` 获取宠物主人ID
