# 对怪防御功能说明

## 一、功能概述

新增两个对怪防御相关属性，用于减少怪物对玩家造成的伤害：

| 属性ID | 属性名称 | 类型 | 说明 |
|--------|---------|------|------|
| **56** | 对怪防御 | 数值类型 | 每增加1点，怪物攻击玩家时伤害减少1点 |
| **116** | 受怪减伤 | 万分比类型 | 每增加100点，怪物攻击玩家时伤害减少1% |

**特性：**
- 仅对怪物攻击玩家时生效（PVE防御）
- 对PVP无效
- 两个属性可以叠加使用

---

## 二、实现方式

通过 **QFunction-0.lua** 中的 `m_base` 函数实现（怪物技能伤害公式函数）。

当 `SkillEffect.xls` 的 Fumula 列配置为 `base` 时，怪物攻击会触发 `m_base` 函数，在其中计算对怪防御和受怪减伤。

### 2. m_base 函数实现

**文件**: `Market_Def/QFunction-0.lua`

```lua
function m_base(actor, target, effectid, skillid, skilllv, race)
    local result = BattleManager:Do(actor, target, effectid, skillid, skilllv, race)
    GameEvent.push(EventCfg.onAttack, actor, target, effectid, skillid, skilllv, race)

    -- 宠物血条处理...
    result = SpeHarmMain(actor, target, result)

    -- 对怪减伤和对怪防御（仅当目标是玩家时生效）
    if isplayer(target) then
        -- 对怪防御（属性56）：固定值减免
        local pveDef = tonumber(abil(target, 56)) or 0
        if pveDef > 0 then
            result = math.max(1, result - pveDef)
        end

        -- 受怪减伤（属性116）：万分比减免
        local reducePct = tonumber(abil(target, 116)) or 0
        if reducePct > 0 then
            local reduced = math.floor(result * reducePct / 10000)
            result = math.max(1, result - reduced)
        end
    end

    return result
end
```

## 四、伤害计算顺序

当怪物攻击玩家时，伤害计算顺序：

```
1. BattleManager:Do 计算基础伤害
2. ↓
3. SpeHarmMain 处理特殊伤害
4. ↓
5. 对怪防御减免（属性56，固定值）
   伤害 = 伤害 - 属性56值
6. ↓
7. 受怪减伤（属性116，万分比）
   伤害 = 伤害 - (伤害 × 属性116 / 10000)
8. ↓
9. 最终伤害（最小为1）
```

## 五、使用示例

### 示例1：对怪防御（属性56）

```
假设：
- 怪物基础伤害：500
- 属性56（对怪防御）：200

计算：
最终伤害 = 500 - 200 = 300
```

### 示例2：受怪减伤（属性116）

```
假设：
- 怪物基础伤害：500
- 属性116（受怪减伤）：3000（万分比，即30%）

计算：
减少伤害 = 500 × 3000 / 10000 = 150
最终伤害 = 500 - 150 = 350
```

### 示例3：两者叠加

```
假设：
- 怪物基础伤害：500
- 属性56（对怪防御）：200
- 属性116（受怪减伤）：3000（30%）

计算：
对怪防御减免 = 500 - 200 = 300
受怪减伤 = 300 × 3000 / 10000 = 90
最终伤害 = 300 - 90 = 210
```

## 六、使用方法

### 方式1：通过装备属性

```lua
-- 在装备属性中添加
{ AttrId = 56, Value = 200 }   -- +200对怪防御
{ AttrId = 116, Value = 3000 } -- +30%受怪减伤
```

### 方式2：通过Buff添加

```lua
-- Buff配置
[600] = {
    ID = 600,
    Name = "守护之力",
    Param = "56#200|116#3000",  -- +200对怪防御，+30%受怪减伤
    Duration = 600,
}
```

### 方式3：通过脚本动态添加

```lua
-- 增加对怪防御
changeabil(actor, 56, "+", 100)

-- 设置受怪减伤（万分比）
changeabil(actor, 116, "=", 5000)  -- 50%减伤

-- 查看属性
local pveDef = abil(actor, 56)
local pveReduce = abil(actor, 116)
print(string.format("对怪防御:%d, 受怪减伤:%.1f%%", pveDef, pveReduce/100))
```

## 七、注意事项

1. **仅对怪物攻击生效**
   - 使用 `isplayer(target)` 判断目标是否是玩家
   - 只对玩家被怪物攻击时生效

2. **伤害最小值**
   - 使用 `math.max(1, damage)` 确保伤害至少为1
   - 避免出现0或负数伤害

3. **叠加顺序**
   - 先计算固定减免（属性56）
   - 再计算百分比减免（属性116）
   - 这样收益最大化

4. **万分比说明**
   - 属性116是万分比类型
   - 100 = 1%
   - 1000 = 10%
   - 10000 = 100%（理论上完全免疫）
