# 对怪防御功能说明

## 一、功能概述

新增两个对怪防御相关属性，用于减少怪物对玩家造成的伤害：

| 属性ID | 属性名称 | 类型 | 说明 |
|--------|---------|------|------|
| **56** | 对怪防御 | 数值类型 | 每增加1点，怪物攻击玩家时伤害减少1点 |
| **116** | 受怪减伤 | 万分比类型 | 每增加100点，怪物攻击玩家时伤害减少1% |

**特性：**
- 仅对怪物攻击玩家时生效（PVE防御）
- 对PVP无效
- 两个属性可以叠加使用

### 2. 被动技能模板

**文件**: `SkillFormula/Custom/CustomPassiveTemplate.lua`

#### DEF_PVE_FLAT_FN - 对怪固定防御

```lua
_T.DEF_PVE_FLAT_FN = {
    fn = function(cfg)
        local attrId = tonumber(cfg.Param1) or 56
        return function(att, tar, sid, ctx)
            -- 当怪物攻击玩家时生效
            if ctx.attPL then
                return  -- 攻击者是玩家，不触发
            end

            local pveDef = abil(tar, attrId) or 0
            if pveDef <= 0 then
                return
            end

            -- 从伤害中减去防御值
            ctx.damage = math.max(1, (ctx.damage or 0) - pveDef)
        end
    end,
}
```

#### DMG_PVE_REDUCE_FN - 受怪减伤（万分比）

```lua
_T.DMG_PVE_REDUCE_FN = {
    fn = function(cfg)
        local attrId = tonumber(cfg.Param1) or 116
        return function(att, tar, sid, ctx)
            -- 当怪物攻击玩家时生效
            if ctx.attPL then
                return  -- 攻击者是玩家，不触发
            end

            local reducePct = abil(tar, attrId) or 0
            if reducePct <= 0 then
                return
            end

            -- 万分比计算减少伤害
            local reduced = math.floor((ctx.damage or 0) * reducePct / 10000)
            ctx.damage = math.max(1, (ctx.damage or 0) - reduced)
        end
    end,
}
```

## 三、伤害计算顺序

当怪物攻击玩家时，伤害计算顺序：

```
1. 基础伤害计算（怪物攻击力 - 玩家防御）
2. ↓
3. 对怪防御减免（属性56，固定值）
   伤害 = 伤害 - 属性56值
4. ↓
5. 受怪减伤（属性116，万分比）
   伤害 = 伤害 - (伤害 × 属性116 / 10000)
6. ↓
7. 最终伤害（最小为1）
```

## 四、使用示例

### 示例1：对怪防御（属性56）

```
假设：
- 怪物攻击力：500
- 玩家基础防御：200
- 属性56（对怪防御）：100

计算：
基础伤害 = 500 - 200 = 300
对怪防御减免 = 300 - 100 = 200
最终伤害 = 200
```

### 示例2：受怪减伤（属性116）

```
假设：
- 怪物攻击力：500
- 玩家基础防御：200
- 属性116（受怪减伤）：2000（万分比，即20%）

计算：
基础伤害 = 500 - 200 = 300
受怪减伤 = 300 × 2000 / 10000 = 60
最终伤害 = 300 - 60 = 240
```

### 示例3：两者叠加

```
假设：
- 怪物攻击力：500
- 玩家基础防御：200
- 属性56（对怪防御）：100
- 属性116（受怪减伤）：2000（20%）

计算：
基础伤害 = 500 - 200 = 300
对怪防御减免 = 300 - 100 = 200
受怪减伤 = 200 × 2000 / 10000 = 40
最终伤害 = 200 - 40 = 160
```

## 五、使用方法

### 方式1：通过装备属性

```lua
-- 在装备属性中添加
{ AttrId = 56, Value = 100 }   -- +100对怪防御
{ AttrId = 116, Value = 2000 } -- +20%受怪减伤
```

### 方式2：通过Buff添加

```lua
-- Buff配置
[600] = {
    ID = 600,
    Name = "守护之力",
    Param = "56#200|116#3000",  -- +200对怪防御，+30%受怪减伤
    Duration = 600,
}
```

### 方式3：通过脚本动态添加

```lua
-- 增加对怪防御
changeabil(actor, 56, "+", 100)

-- 设置受怪减伤（万分比）
changeabil(actor, 116, "=", 5000)  -- 50%减伤

-- 查看属性
local pveDef = abil(actor, 56)
local pveReduce = abil(actor, 116)
sendmsg(actor, 9, string.format("对怪防御:%d, 受怪减伤:%.1f%%", pveDef, pveReduce/100))
```

## 六、注意事项

1. **仅对怪物攻击生效**
   - ctx.attPL 判断攻击者类型
   - true = 玩家攻击（不触发）
   - false = 怪物攻击（触发）

2. **伤害最小值**
   - 使用 `math.max(1, damage)` 确保伤害至少为1
   - 避免出现0或负数伤害

3. **叠加顺序**
   - 先计算固定减免（属性56）
   - 再计算百分比减免（属性116）
   - 这样收益最大化

4. **万分比说明**
   - 属性116是万分比类型
   - 100 = 1%
   - 1000 = 10%
   - 10000 = 100%（理论上完全免疫）

5. **与属性165（对怪伤害）的区别**
   - 属性165：玩家攻击怪物时的**伤害加成**
   - 属性56/116：怪物攻击玩家时的**伤害减免**
   - 一个进攻，一个防御
