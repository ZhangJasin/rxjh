# 对怪伤害属性165 使用说明

## 一、功能概述

属性ID 165 是对怪伤害加成属性，每增加1点，攻击怪物时最终伤害增加1点。

**特性：**
- 仅对怪物生效，对玩家无效（PVE专属）
- 固定伤害加成，在伤害计算最后阶段追加
- 全职业通用
- 与百分比增伤叠加（先计算百分比，再加固定值）

### 2. 被动技能模板
**文件**: `SkillFormula/Custom/CustomPassiveTemplate.lua`

```lua
_T.DMG_PVE_FLAT_FN = {
    fn = function(cfg)
        local attrId = tonumber(cfg.Param1) or 165
        local ratio = tonumber(cfg.Param2) or 1
        return function(att, tar, sid, ctx)
            -- 仅对怪物生效
            if ctx.attPL then
                return
            end
            
            local pveAttr = abil(att, attrId) or 0
            if pveAttr <= 0 then
                return
            end
            
            local pveDamage = math.floor(pveAttr * ratio + 0.5)
            ctx.damage = (ctx.damage or 0) + pveDamage
        end
    end,
}
```

## 三、使用方法

### 方式1：通过装备属性

在装备、宝石、附魔等任何可以添加属性的地方，直接添加属性ID 165：

**示例 - 装备属性配置（Item.lua或相关配置）：**
```lua
-- 在某件装备的属性中添加
{
    AttrId = 165,      -- 对怪伤害
    Value = 50,        -- +50对怪伤害
}
```

**效果：** 穿戴该装备攻击怪物时，最终伤害+50

### 方式2：通过Buff添加

在Buff配置中添加属性165：

**示例 - Buff配置（Buff.lua）：**
```lua
[500] = {
    ID = 500,
    Name = "猎魔之力",
    Param = "165#100",  -- 属性165，值100
    Duration = 300,     -- 持续300秒
}
```

**效果：** 获得Buff后，攻击怪物时伤害+100，持续5分钟

### 方式3：通过脚本动态添加

在游戏脚本中动态增加属性：

```lua
-- 给玩家增加100点对怪伤害
changeabil(actor, 165, "+", 100)

-- 设置玩家对怪伤害为50
changeabil(actor, 165, "=", 50)

-- 查看玩家对怪伤害
local pveDmg = abil(actor, 165)
sendmsg(actor, 9, "你的对怪伤害: " .. pveDmg)
```

### 方式4：通过宝石系统

在宝石属性配置中添加（参考现有的宝石系统）：

**示例 - 在宝石配置文件中：**
```lua
-- 新增一种"猎魔石"
{
    GemId = 1500,
    Name = "猎魔石",
    AttrId = 165,      -- 对怪伤害
    MinValue = 10,     -- 最小值
    MaxValue = 30,     -- 最大值
}
```

## 四、伤害计算示例

### 示例1：基础对怪伤害

```
假设：
- 基础伤害：1000
- 目标防御：200
- 对怪伤害属性165：50

计算：
防御减免后伤害 = 1000 - 200 = 800
最终伤害 = 800 + 50 = 850
```

### 示例2：与百分比增伤叠加

```
假设：
- 基础伤害：1000
- 目标防御：200
- 对怪增伤115：20%（万分比2000）
- 对怪伤害165：50

计算：
防御减免后伤害 = 1000 - 200 = 800
百分比增伤 = 800 × (1 + 2000/10000) = 800 × 1.2 = 960
固定伤害加成 = 960 + 50 = 1010
```

### 示例3：暴击+对怪伤害

```
假设：
- 基础伤害：1000
- 目标防御：200
- 触发暴击（会心伤害58：5000）
- 对怪伤害165：50

计算：
防御减免后伤害 = 1000 - 200 = 800
暴击倍率 = 1.5 + 5000/10000 = 2.0
暴击伤害 = 800 × 2.0 = 1600
最终伤害 = 1600 + 50 = 1650
```

## 五、配置调整

### 修改伤害系数

如果想让1点属性165 = 2点伤害，修改Passive.xls：

```
Param2 = 2  （原来是1）
```

或者修改模板中的默认值：
```lua
local ratio = tonumber(cfg.Param2) or 2  -- 改为2
```

### 修改触发概率

当前是100%触发（每次攻击都生效）。如果想改为概率触发：

**修改Passive.xls：**
```
BaseProb = 0.5  （50%概率触发）
```

**修改模板：**
```lua
_T.DMG_PVE_FLAT_FN = {
    fn = function(cfg)
        local attrId = tonumber(cfg.Param1) or 165
        local ratio = tonumber(cfg.Param2) or 1
        local prob = tonumber(cfg.BaseProb) or 1  -- 添加概率
        return function(att, tar, sid, ctx)
            if ctx.attPL then return end
            if math.random() < prob then  -- 概率判定
                local pveAttr = abil(att, attrId) or 0
                if pveAttr <= 0 then return end
                local pveDamage = math.floor(pveAttr * ratio + 0.5)
                ctx.damage = (ctx.damage or 0) + pveDamage
            end
        end
    end,
}
```

## 六、注意事项

1. **仅对怪物生效**
   - 对玩家PVP无效
   - ctx.attPL 用于判断目标类型（true=玩家，false=怪物）

2. **伤害计算顺序**
   - 在基础伤害、暴击、百分比增伤之后追加
   - 公式：`最终伤害 = (基础伤害 - 防御) × 暴击倍率 × 百分比增伤 + 固定伤害165`

3. **属性获取方式**
   - `abil(att, 165)` - 获取角色属性165的总值
   - 包括装备、Buff、被动技能等所有来源的累加

4. **与其他增伤的关系**
   - 与属性115（对怪增伤）叠加：115是百分比，165是固定值
   - 与属性109（追击伤害）叠加：都是固定伤害，直接相加

5. **显示问题**
   - 属性165默认Isshow=1，会在角色面板显示
   - 如需隐藏，改为Isshow=0

## 八、扩展应用

### 1. 职业专属对怪伤害

为不同职业设置不同的对怪伤害加成：

**在Passive.xls中添加多行：**

```
ID=100201, JobReq=1, TemplateName=DMG_PVE_FLAT_FN, Param1=165, Param2=1  (弓手)
ID=100202, JobReq=2, TemplateName=DMG_PVE_FLAT_FN, Param1=165, Param2=1  (枪客)
ID=100203, JobReq=3, TemplateName=DMG_PVE_FLAT_FN, Param1=165, Param2=1  (刺客)
...
```

### 2. 技能等级影响对怪伤害

根据技能等级动态调整：

```lua
_T.DMG_PVE_SKILL = {
    fn = function(cfg)
        local skillId = tonumber(cfg.Param1) or 0
        local ratio = tonumber(cfg.Param2) or 1
        return function(att, tar, sid, ctx)
            if ctx.attPL then return end
            local skillLv = skilllv(att, skillId) or 0
            local pveDamage = skillLv * ratio
            ctx.damage = (ctx.damage or 0) + pveDamage
        end
    end,
}
```

### 3. 根据怪物类型差异化

根据不同怪物类型给予不同加成：

```lua
_T.DMG_PVE_BY_MONSTER_TYPE = {
    fn = function(cfg)
        local monsterType = tonumber(cfg.Param1) or 0
        local bonus = tonumber(cfg.Param2) or 0
        return function(att, tar, sid, ctx)
            if ctx.attPL then return end
            if raceinfo(tar, "TYPE") == monsterType then
                ctx.damage = (ctx.damage or 0) + bonus
            end
        end
    end,
}
```
