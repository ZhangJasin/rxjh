## 一、伤害公式系统架构

### 1.1 核心组件

```
伤害计算系统
├── BattleManager.lua      (二进制) - 战斗管理器，核心伤害计算引擎
├── PassiveManager.lua     (二进制) - 被动技能管理器
├── QiGongManager.lua      (源码)   - 气功管理器
│
├── DamageFn.xls           - 伤害公式Excel配置表(二进制)
├── HealFn.xls             - 治疗公式Excel配置表(二进制)
├── Passive.xls            - 被动技能公式Excel配置表(二进制)
│
└── Custom/
    ├── CustomConst.lua    - 战斗常量定义
    ├── CustomFuncs.lua    - 自定义工具函数
    └── CustomPassiveTemplate.lua - 被动技能模板(2408行)
```

### 1.2 数据流向

```
玩家攻击 → QFunction-0.lua (事件入口)
         ↓
    GameEvent.push(EventCfg.onAttack, ...)
         ↓
    BattleManager 创建战斗上下文 ctx
         ↓
    加载被动模板 → 执行事件回调
         ↓
    读取 DamageFn.xls 公式配置
         ↓
    计算最终伤害 → 应用到目标
```

## 三、伤害计算公式推导

### 3.1 基础伤害公式

根据代码分析，基础伤害计算遵循以下流程：

```lua
-- 伪代码推导
基础伤害 = 攻击力(23) × 技能系数

-- 考虑武功伤害
武功伤害 = 基础伤害 × (1 + 武功攻击(53) / 系数)

-- 考虑防御减伤
最终伤害 = 武功伤害 - 目标防御(52) × (1 - 忽视防御比例)
```

### 3.2 会心一击(暴击)公式

```lua
-- 暴击概率计算
暴击概率 = max(0.10, min(0.90, (攻击者会心几率(57) - 目标会心抵抗(59)) / 10000))

-- 暴击伤害
暴击倍率 = 1.5 + 攻击者会心伤害(58) / 10000
最终伤害 = 基础伤害 × 暴击倍率
```

### 3.3 必杀一击公式

```lua
-- 必杀概率 = 基础概率 + 被动加成(s.PROB.bisha)
-- 必杀伤害倍率固定为 1.5×
ctx.mul = 1.5 + extraCrit
ctx.pzDamage = CustomConst.AttackPiaoZi.BISHA  -- 飘字ID 11
```

### 3.4 超必杀(心神凝聚)公式

```lua
-- 超必杀概率 = 基础概率 + 被动加成(s.PROB.cbs)
-- 超必杀伤害倍率
基础倍率 = 2.0
额外倍率 = s.MUL.cbs_inc  -- 被动累加的百分比
最终倍率 = 基础倍率 × (1 + 额外倍率)
ctx.mul = 最终倍率
ctx.pzDamage = CustomConst.AttackPiaoZi.XINSHEN  -- 飘字ID 10 (2×)
```

### 3.5 PVP伤害修正公式

```lua
-- PVP环境下的伤害增益
PK伤害加成 = 攻击者PK加成(67) / 10000
PK伤害减免 = 防御者PK减免(68) / 10000

PVP最终伤害 = 原始伤害 × (1 + PK伤害加成) × (1 - PK伤害减免)
```

### 3.6 PVE伤害修正公式

```lua
-- PVE环境下的伤害增益
对怪增伤 = 攻击者对怪增伤(115) / 10000
受怪减伤 = 防御者受怪减伤(116) / 10000

PVE最终伤害 = 原始伤害 × (1 + 对怪增伤) × (1 - 受怪减伤)
```

## 四、被动技能伤害模板详解

### 4.1 怒气系统伤害模板

#### RAGE_ADD - 被动触发加怒气

```lua
-- 触发条件: 造成伤害 > 0
-- 计算公式
基础增加值 = cfg.Param2
怒气增加量 = floor(基础增加值 × (1 + s.RAGE.addpct) + 0.5)
怒气增加量 += floor(1000 × 攻击者怒气属性(112) / 10000)

-- 特殊机制: 目标有清怒气几率
if math.random(1,10000) <= 目标清怒气几率(111) then
    设置怒气 = 0
    return
end
```

#### RAGE_FULL - 满怒Buff属性

```lua
-- 满怒时触发Buff持续 baseDur 秒
-- 额外百分比来自气功: extraPct = s.QG.kfwp
实际加成值 = floor(配置值 × (1 + extraPct) + 0.5)
```

### 4.2 连击点系统(刺客专属)

```lua
-- 连击点上限 = 5
-- 满怒时连击点 +1
if 当前连击点 < 5 then
    连击点 += 1
end

-- 技能获得连击点
释放武功技能 → 连击点 += cfg.Param2
```

### 4.3 伤害加成模板

#### DMG_ADD - 静态伤害加成

```lua
-- 写入静态缓存 s.QG
键名 = cfg.Param1 (默认 "incDamage")
增加值 = cfg.Param2 × 2  -- 注意这里乘以2
s.QG[键名] += 增加值
```

#### DMG_WG_PCT - 武功伤害百分比

```lua
-- 写入 s.QG.wgpct
键名 = cfg.Param1 (默认 "wgpct")
s.QG[键名] += cfg.Param2
```

#### DMG_PADD - 动态伤害加成

```lua
-- 概率触发，临时加成
if rand() < 基础概率 then
    ctx.pctDamage += cfg.Param2
end
```

### 4.4 特殊伤害模板

#### 反伤触发 (DMG_FS)

```lua
-- 触发概率区分PVP/PVE
触发概率 = isPlayer and probPlayer or probMonster

if rand() < 触发概率 then
    反伤伤害 = floor(受到伤害 × 反伤比例 + 0.5)
    目标扣血 = 反伤伤害
end
```

#### 防御转化伤害 (DMG_BY_DEF)

```lua
-- 将目标防御按比例转化为伤害
if rand() < 基础概率 then
    目标防御 = ctx.D(防御属性ID)
    额外伤害 = floor(目标防御 × 转化比例 + 0.5)
    ctx.damage += 额外伤害
end
```

#### 攻击转化伤害 (DMG_BY_ATK)

```lua
-- 将自身攻击属性按比例转化为伤害
if rand() < 基础概率 then
    攻击值 = ctx.A(攻击属性ID)
    额外伤害 = floor(攻击值 × 转化比例 + 0.5)
    ctx.damage += 额外伤害
end
```

#### 武功伤害转化 (DMG_BY_WG)

```lua
-- 仅对技能攻击生效
if ctx.is_skill and ctx.damage > 0 then
    if rand() < 基础概率 then
        额外伤害 = floor(ctx.damage × 转化比例 + 0.5)
        ctx.damage += 额外伤害
    end
end
```

#### 闪避反击伤害 (DMG_BY_DODGE)

```lua
-- 对方闪避后，记录伤害到跨帧缓存
基础伤害 = ctx.curDamage
额外伤害 = floor(基础伤害 × 比例 + 0.5)
BattleManager._carryDamage[闪避者] = 额外伤害

-- 下次攻击时追加伤害
ctx.damage += BattleManager._carryDamage[攻击者]
BattleManager._carryDamage[攻击者] = nil  -- 用后清除
```

### 4.5 吸血功能

```lua
-- 在CustomFuncs.lua中实现
吸血值 = env.A(114)  -- 属性ID 114: 打怪吸血
if 吸血值 > 0 then
```

### 4.6 无坚不摧 (减防机制)

```lua

-- 忽视防御 = 基础比例 + 被动加成(s.CUSTOM.wjbc_ignore)

    减少防御 = floor(目标防御值 × 最终比例 + 0.5)

end
```

## 五、刺客武功技能伤害机制

### 5.1 通用机制

所有刺客武功技能(金鸡独立、猛虎隐林、拨草寻蛇、流云突刺、聚气凝神、御剑飞虹)都遵循以下模式：

```lua
-- 1. 检查伤害是否有效
if not (ctx.damage and ctx.damage > 0) then return end

-- 2. 检查连击点
当前连击点 = currabil(att, comboId)
if 当前连击点 < 1 then return end

-- 3. 概率免费释放
免费概率 = s.PROB.combo_ignore
if rand() >= 免费概率 then
    changeabil(att, comboId, "-", 1)  -- 消耗连击点
end

-- 4. 增加超必杀概率
s.PROB.cbs += CBSProb  -- cfg.Param3/100

-- 5. 添加Buff到目标
addbuff(目标, buffId, buffdur)

-- 6. 记录伤害到Buff自定义数据
setbuffcustdata(目标, buffId, floor(ctx.damage))

-- 7. 解除怒气锁定
s.RAGE.lock = false
```

### 5.2 特殊技能机制

#### 猛虎隐林 / 聚气凝神

```lua
-- 特殊: 扣除目标血量百分比
扣血 = floor(ctx.damage × delhpPct / 100)
扣血 = max(1, 扣血)  -- 至少1点
setbuffcustdata(目标, buffId, "" .. 扣血)
```

## 九、实战伤害计算示例

### 9.1 普通攻击伤害

```
假设:
- 攻击力: 1000
- 目标防御: 200
- 忽视防御: 10% (1000/10000)
- 无暴击

计算:
基础伤害 = 1000
防御减免 = 200 × (1 - 0.10) = 180
最终伤害 = 1000 - 180 = 820
```

### 9.2 暴击伤害

```
假设:
- 攻击力: 1000
- 目标防御: 200
- 会心几率: 2000 (20%)
- 目标会心抵抗: 500 (5%)
- 会心伤害: 5000 (50%)

计算:
暴击概率 = max(0.10, min(0.90, (2000-500)/10000)) = 15%
if 触发暴击:
    暴击倍率 = 1.5 + 5000/10000 = 2.0
    最终伤害 = (1000 - 180) × 2.0 = 1640
```

### 9.3 超必杀伤害

```
假设:
- 攻击力: 1000
- 目标防御: 200
- 超必杀触发
- 被动增伤: 20% (s.MUL.cbs_inc = 0.2)

计算:
超必杀倍率 = 2.0 × (1 + 0.2) = 2.4
最终伤害 = (1000 - 180) × 2.4 = 1968
```

### 9.4 PVP伤害

```
假设:
- 基础伤害: 1000
- PK加成: 500 (5%)
- 目标PK减免: 300 (3%)

计算:
PVP伤害 = 1000 × (1 + 0.05) × (1 - 0.03) = 1000 × 1.05 × 0.97 = 1018.5
```

### 9.5 反伤伤害

```
假设:
- 受到攻击伤害: 500
- 反伤比例: 30% (cfg.Param2 = 0.3)
- 触发概率: 20%

if rand() < 0.2:
    反伤 = floor(500 × 0.3) = 150
    攻击者扣血150
```

## 十四、总结与优化建议

### 14.1 伤害公式核心要点

1. **基础伤害** = 攻击力 - 防御 × (1 - 忽视防御比例)
2. **暴击系统** = 概率触发 × 倍率伤害
3. **必杀/超必杀** = 高概率触发 × 高倍率伤害
4. **PVP/PVE** = 独立加减乘算
5. **被动模板** = 静态缓存 + 动态触发
6. **怒气系统** = 资源管理，满怒触发爆发

### 14.2 伤害优化建议

#### 攻击向

1. **优先级**: 攻击力 > 武功伤害 > 追击伤害 > 会心伤害
2. **暴击流**: 堆会心几率 + 会心伤害
3. **必杀流**: 堆必杀概率 + 超必杀概率
4. **PVP特化**: 堆对人战斗 + PK加成

#### 防御向

1. **优先级**: 生命 > 防御 > 武功防御 > 伤害减免
2. **闪避流**: 堆回避 + 闪避几率
3. **抵抗流**: 堆会心抵抗 + 对人抵抗
4. **PVP特化**: 堆PK减免 + 伤害减免

### 14.3 公式扩展性

- 通过 `CustomPassiveTemplate.lua` 可自定义新被动技能
- 通过 `DamageFn.xls` 可调整伤害计算步骤
- 通过 `Passive.xls` 可配置新被动技能参数
- 通过 `CustomFuncs.lua` 可添加自定义函数
