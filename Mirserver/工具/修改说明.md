# 富文本编辑器修改说明

## 最新更新 (2026-04-18)

### 新增功能

#### 1. **编辑模式选择** (UBB/HTML)
- ✅ 在底部工具栏添加了编辑模式选择
- **UBB模式**: 使用 `[size=50]`, `[color=red]`, `[b]`, `[i]`, `[u]`, `[br]` 等标签
- **HTML模式**: 使用 `<font size=50>`, `<font color=red>`, `<b>`, `<i>`, `<u>`, `<br>` 等标签
- 切换模式时会提示当前已选择的模式
- 所有格式化操作都会根据当前模式生成对应的标签

#### 2. **底部工具栏区域**
所有编辑操作现在都集中在窗口下方的工具栏:

**第一行 - 模式选择和格式化工具:**
- 编辑模式选择: UBB / HTML 单选按钮
- 格式化工具: **B** (粗体) / *I* (斜体) / U̲ (下划线)
- 同步按钮: 
  - `←同步` - 从预览区生成代码到编辑区
  - `同步到→` - 从编辑区渲染到预览区

**第二行 - 字号和颜色:**
- 字号选择器 (10-80)
- 应用字号按钮
- 自定义颜色选择器
- 刷新预览按钮

**第三行 - 格式提示:**
- 根据当前模式显示对应的格式说明文字

## 之前的修改内容

### 1. 编译文件名修改
- ✅ 将编译后的文件名从 `Item备注修改工具.exe` 改为 `富文本编辑器.exe`
- 修改文件：`Item备注修改工具.spec`

### 2. 支持双格式富文本
- ✅ 编辑区同时支持 **UBB** 和 **HTML** 两种富文本语法
- **UBB格式**：`[size=50]大字[/size]` `[color=red]红色[/color]` `[br]` `[b]` `[i]` `[u]`
- **HTML格式**：`<font size=50>大字</font>` `<font color=red>红色</font>` `<br>` `<b>` `<i>` `<u>`
- 系统会自动检测当前文本使用的格式

### 3. 预览区支持双格式
- ✅ 预览区同样支持 UBB 和 HTML 两种格式的展示
- 自动检测格式并渲染：
  - 如果文本包含HTML标签，使用HTML解析器
  - 如果文本包含UBB标签，使用UBB解析器
  - 纯文本则直接显示
- HTML预览器同时兼容UBB标签（混用场景）

### 4. 编辑区选中效果优化
- ✅ 在编辑区选中文本后，不会立即清除选中效果
- ✅ 只有在点击"应用字号"或"自定义颜色"按钮并应用后，才会清除选中
- 这样用户可以清楚地看到自己选中的内容

## 主要功能

### 格式检测
```python
detect_format(text)  # 返回 'html', 'ubb', 或 'plain'
```

### 格式渲染
- `_render_ubb_preview()` - 渲染UBB格式预览
- `_render_html_preview()` - 渲染HTML格式预览（同时兼容UBB标签）

### 智能应用
- `_apply_editor_format()` - 根据当前模式应用粗体/斜体/下划线
- `_apply_size()` - 根据当前模式应用字号标签
- `_apply_custom_color()` - 根据当前模式应用颜色标签

### 双向同步
- `_sync_to_preview()` - 从编辑区同步到预览区
- `_sync_from_preview()` - 从预览区反向生成代码到编辑区

## 使用说明

1. **打开文件**：支持 `.xls` 和 `.xlsx` 格式
2. **选择物品**：在左侧列表搜索并选择要编辑的物品
3. **选择编辑模式**：
   - 在底部工具栏选择 **UBB** 或 **HTML** 模式
   - 切换后所有操作都会生成对应格式的标签
4. **编辑备注** (两种方式):
   
   **方式一：代码编辑**
   - 在右侧"编辑源码"区手动输入标签
   - 使用底部工具栏的格式化按钮快速添加标签
   - 左侧实时预览渲染效果
   
   **方式二：可视化编辑**
   - 在左侧"可视化预览"区直接编辑文字
   - 应用格式后点击"←同步"生成代码
5. **保存备注**：点击"保存备注"按钮保存到原文件

## 支持的标签对照表

| 效果 | UBB格式 | HTML格式 |
|------|---------|----------|
| 字号 | `[size=50]文字[/size]` | `<font size=50>文字</font>` |
| 颜色 | `[color=red]文字[/color]` | `<font color=red>文字</font>` |
| 粗体 | `[b]文字[/b]` | `<b>文字</b>` |
| 斜体 | `[i]文字[/i]` | `<i>文字</i>` |
| 下划线 | `[u]文字[/u]` | `<u>文字</u>` |
| 换行 | `[br]` | `<br>` 或 `<br/>` |

## 技术细节

- 底部工具栏集中所有编辑操作
- UBB/HTML模式切换，方便用户选择喜欢的格式
- 自动根据模式生成对应的标签代码
- 预览区支持混合格式渲染
- 所有操作直接保存回原Excel文件
